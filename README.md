# Exercícios de Programação - Faculdade

Este repositório reúne exercícios e atividades desenvolvidas durante a graduação em **Sistemas de Informação**.  
Os códigos foram criados com o objetivo de praticar **lógica de programação, estruturas de dados e conceitos fundamentais de desenvolvimento de software**, utilizando diferentes linguagens de programação.

## Linguagens e Tecnologias

- C#
- Java
- Python
- Kotlin
- Banco de Dados SQL

## Estrutura do repositório

csharp → exercícios desenvolvidos em C#  
java → exercícios desenvolvidos em Java  
python → exercícios desenvolvidos em Python  
kotlin → exercícios desenvolvidos em Kotlin  
sql → exercícios e modelagem de banco de dados

Cada pasta contém pequenos programas e atividades propostas durante as disciplinas da graduação.

## Objetivo

Praticar e consolidar conhecimentos em:

- Lógica de programação
- Estruturas condicionais e de repetição
- Manipulação de listas e coleções
- Entrada e saída de dados
- Estruturas básicas de dados
- Programação orientada a objetos

## Exemplos de exercícios

- Lista de números utilizando **List em C#**
- Sistema simples de biblioteca em **Kotlin**
- Exercícios básicos de lógica em **Java**
- Manipulação de listas e algoritmos em **Python**
- Projeto de modelagem de banco de dados relacional em **SQL**
